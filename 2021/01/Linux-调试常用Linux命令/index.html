    进程线程状态

    top

    Linux下的任务管理器

    1 察看多核心CPU各个核心的情况

P 按照CPU使用率排行
T 按照CPU使用时间排行
M 按照物理内存使用率排行

top -p xx 察看指定pid的信息

    htop

    更加现代的top 有颜色 支持鼠标 杀进程极其方便

    sudo yum install epel-release

sudo yum install htop

    strace

    跟踪进程运行时使用的系统调用和信号

    -t          每行前加上输出信息

-tt         the time printed will include the microseconds.

-ttt        开头部分将打印为自该纪元以来的秒数。 the time printed will include the microseconds

-T          显示系统调用所需时间

-o          输出文件

-p          指定pid

-f -F       跟踪fork和vfork

    strace -T -tt -p $(lsof -i tcp:4000 -t) -o /root/live.log

    tcpdump

    -i 指定网卡

    -nn 不解析ip和port 显示原本的内容

    -S 绝对值显示偏移量

    -vv 显示详细的抓包数据

    -vvv 更详细的抓包数据

    -w 将结果保存到文件 可以通过wireshark观看
